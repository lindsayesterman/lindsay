Project Finder

Project Finder helps developers find project ideas to build up their portfolios. Anyone can post web app ideas and devs can sort through them until they find one they like.

I made this app because I like spending my time solving actual problems. It can be hard to do user research and come up with useful ideas. Project Finder allows for more communication and classification of ideas. Check out the site at project-finder.vercel.app! Stack: React, CSS, Node, Express, PostgreSQL.
