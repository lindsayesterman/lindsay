AP Class Help

Website dedicated to helping students in AP classes learn and review material on their own. These are some of the resources that have been most helpful to me and there are extra ones my friends recommended. Taking many APs can be overwhelming, so a nice list to look back to when stuck on a topic can be really helpful. Check out the site at apclasshelp.com.

Check out the github repo here.


National Park Finder

Enter a state abbreviation and a list of national parks in your area will appear! Helps folks plan a vacation or nice weekend. This website is the first time I used an API. Check out National Park finder to get a better idea of the site!

Check out the github repo here.


Noteful

Noteful is an app dedicated to productivity. You can create your own list of to do items, categorize them, and delete when done.

Check out the github repo here.


Dog Checklist

The first website I ever created! I built this to help people when they get a new dog and are overwhelmed by what to do. There is a checklist so users can check off items as they finish them. Also included are highly rated options of what is recommended. Check out dogchecklist.org!

Check out the github repo here.

Stop Gerrymandering

Website I created to encourage the ban of gerrymandering. Users can look up their local representative and send a pre-written (or reworked if they want) letter to the congress representative. The immorality of gerrymandering has been something I was passionate about ever since I learned about it. When starting web dev, I knew I wanted to use it to try and solve problems like this. Check out the website!
